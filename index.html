If you are comfortable with graphing and have perhaps dabbled in a bit of Calculus I, then feel free to continue on to the rest of the site. But if it has been some time since you were forced to learn this stuff, below is a brief overview of mathematical functions and how they are graphed.

Graphing 101

A graph is a drawing that illustrates an algebraic operation or equation in a two-dimensional plane. A graph allows you to see the characteristics of an algebraic statement immediately.

To graph an algebraic equation, we use a coordinate plane that features two intersecting axes - two perpendicular lines that cross at a point called the origin. The horizontal line is called the X-axis, and the vertical line is called the Y-axis. The scales on the axes are shown with little tick marks. Any given point on the graph is identified by an ordered pair of coordinates. The x-coordinate indicates how far and in which direction the point is from the origin along the X-axis, and the y-coordinate indicates how far from the origin and in which direction the point is along the Y-axis. Coordinates that lie to the right and above the origin are positive, while those to the left and below the origin are negative. A line on a graph is a collection of many points, each one identified by a value for x and a value for y.

Functions

Not to be confused with software functions, a mathematical function is a relationship between two variables that features exactly one output value for every input value - in other words, exactly one y-value for every x-value inserted into the function equation. At every x-value on the graph, the function calculates one y-value, drawing a line that moves across the graph as the x-values move from negative to positive (left to right).

Function Notation

The "f (x)" that is found on the left side of the equals sign, pronounced "F of X", tells us that we should replace the x in the equation on the right side of the equals sign with some value, and then calculate the equation with that value. The result is the y-value at that x-value on the graph. To indicate an exponent, we use the carrot symbol (shift + 6).

For example, suppose we have a function such as f (x) = x^2. If we replace the x in the equation with a value of three, we can rewrite the function like so: f (3) = 3^2. So when x is three, y is nine. With every value of x on the graph, the function calculates a value for y.

Slope

The slope is a number that represents the steepness and direction of a line. The greater the numerical value of the slope, the steeper the line is. If a line's slope is positive, the line rises from left to right. If the slope is negative, the line falls from left to right.

Derivative

The derivative is an independent function that measures the slope of another function. If the first function is written as f (x) = ..., its derivative is written as f ' = ... (pronounced "F prime"). Suppose we have a function whose shape on the graph is a parabola (U-shape). Initially, the slope is negative as the line falls from left to right. As the line begins to turn, the slope gets closer to zero. When the line is exactly flat, the slope is precisely zero, and when the line begins to turn upwards, the slope turns positive.

When the derivative is graphed, the y-value represents the slope of the first graph at each value for x. When the y-value of the derivative graph is negative (below the X-axis), it means the slope of the first function is negative at that x-value on its graph. When the y-value is positive (above the X-axis), it means the slope of the first function is positive at that x-value. And when the y-value is exactly zero, it means the line on the first graph is flat at that x-value. Also, a greater absolute-value of y (moving away from the origin) represents a steeper slope. Therefore, the shape of the derivative graph can measure the rate at which the slope of the first function is changing.
